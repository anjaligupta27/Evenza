# Evenza - College Event Management System

## 📌 Project Description
Evenza is a full-stack college event management system where:
- Admin creates events
- Students register and view events

## 🚀 Technologies Used
- HTML, CSS, JavaScript
- Node.js
- Express.js
- EJS Template Engine
- Middleware
- REST APIs

## ⚙️ Features
- User Registration & Login
- Event Creation (Admin)
- My Events Page
- Middleware Logging
- SSR using EJS
- Cookies Handling

## 🖥️ How to Run

1. Clone repository
2. Go to backend folder
3. Install dependencies
